hexo+github搭建自己的blogs

Author: chenbd03 Date: February 19, 2022  8:44:16 Category: hexo-github搭建blog

Welome to chenbd03’ s blog！个人博客基于windows环境下的hexo + github进行搭建，过程比较简单，来搭建属于自己的博客吧！

  1. 安装相关配置工具Git + nodejs + hexo，并本地启动

    1）在Win10上进入Git工具的官网，可直接从https://git-scm.com/downloads 下载， 选择默认选项安装即可（实在不会可通过百度自寻查询安装！）；安装完成后，在命令行窗口，输入： git –version，查看版本，若查询成功，则证明安装成功

    2）因为hexo是基于Node.js驱动的博客框架，因此需要安装NodeJS： https://nodejs.org/en/download/ （安装过程可参考：https://www.cnblogs.com/liuqiyun/p/8133904.html）；安装完成后，在命令行窗口分别输入：node -v 和 npm -v ， 若显示版本号，则安装成功；

    3）相关环境安装完成，现在通过npm工具开始安装Hexo；在命令行窗口输入：npm install hexo-cli -g，安装完hexo后，新建一个文件夹（例如myblog），右键打开“Git Bash Here”，然后通过在Git命令行下输入： hexo init （看到start blogging with Hexo 就表示初始化成功了）， 待初始化完后，可通过输入： hexo s 命令，在浏览器输入：http://localhost:4000， 回车就能看到预览效果了，这样一个基本的blog就搭建成功了~

  2. 加载相关主题并配置
    1）通过刚搭建的blog都是默认的主题，若想要个性化，可以到hexo官网下载对应的主题：https://hexo.io/themes/， 自己采用的是https://github.com/Siricee/hexo-theme-Chic ，这款有明暗两种颜色。通过在github 将hexo-theme-Chic主题下载下来，放到在myblog/themes中，在myblog下，找到_config.yml 文件，修改为：theme: hexo-theme-Chic（注意和主题文件名一致！）

    2）修改完成后，在myblog下右键选择“Git Bash Here”，执行hexo new page category 和 hexo new page tag， 执行完后系统在myblog/source下多出对应的文件夹，在对应的文件夹下都有一个index.md， 分别打开index.md，对应修改为：

  3. 部署博客到github上

1）首先要有自己的github账号（没有的小伙伴自行注册一个，务必请记住自己的用户名，注册邮箱，和对应的密码，后续有用到！）

2）注册完github后，需要建立一个新的仓库，点击new repository 或 start project， 注意Github 仅能使用一个同名仓库的代码托管一个静态站点，这里注意仓库名一定要是： 用户名.github.io （很重要！！！）
