删除链表的倒数第N个节点

【题目来源】https://leetcode-cn.com/problems/remove-nth-node-from-end-of-list/

给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

示例1：

输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]

示例 2：
输入：head = [1], n = 1
输出：[]

示例 3：
输入：head = [1,2], n = 1
输出：[1]

【解题思路】这道题直接用双指针方式去解决最直接！先让快指针先走n步，然后再同步一起走，细节就是要判断一些边界条件
